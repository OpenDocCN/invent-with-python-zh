# #8 日历制作者

> 原文：<http://inventwithpython.com/bigbookpython/project8.html>

![](img/9d995d63aaead72cad01120081eb8f75.png)

这个程序为你输入的月份和年份生成可打印的月历文本文件。日期和日历在编程中是一个棘手的话题，因为有太多不同的规则来确定一个月中的天数，哪一年是闰年，以及特定日期是星期几。幸运的是，Python 的`datetime`模块为您处理这些细节。这个程序主要是为月历页面生成多行字符串。

## 它是如何工作的

注意，`getCalendarFor()`函数返回给定月份和年份的日历的一个巨大的多行字符串。在这个函数中，`calText`变量存储这个字符串，这个字符串添加了行、空格和日期。为了跟踪日期，`currentDate`变量保存了一个`datetime.date()`对象，通过增加或减少`datetime.timedelta()`对象将其设置为下一个或上一个日期。你可以通过阅读《Python 自动化指南》的第 17 章来了解 Python 的日期和时间模块，[`automatetheboringstuff.com/2e/chapter17`](https://automatetheboringstuff.com/2e/chapter17/)。

```py
"""Calendar Maker, by Al Sweigart email@protected #  1
Create monthly calendars, saved to a text file and fit for printing. #  2
This code is available at https://nostarch.com/big-book-small-python-programming #  3
Tags: short""" #  4
 #  5
import datetime #  6
 #  7
# Set up the constants: #  8
DAYS = ('Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', #  9
       'Friday', 'Saturday') #  10
MONTHS = ('January', 'February', 'March', 'April', 'May', 'June', 'July', #  11
         'August', 'September', 'October', 'November', 'December') #  12
 #  13
print('Calendar Maker, by Al Sweigart email@protected') #  14
 #  15
while True:  # Loop to get a year from the user. #  16
   print('Enter the year for the calendar:') #  17
   response = input('> ') #  18
 #  19
   if response.isdecimal() and int(response) > 0: #  20
       year = int(response) #  21
       break #  22
 #  23
   print('Please enter a numeric year, like 2023.') #  24
   continue #  25
 #  26
while True:  # Loop to get a month from the user. #  27
   print('Enter the month for the calendar, 1-12:') #  28
   response = input('> ') #  29
 #  30
   if not response.isdecimal(): #  31
       print('Please enter a numeric month, like 3 for March.') #  32
       continue #  33
 #  34
   month = int(response) #  35
   if 1 <= month <= 12: #  36
       break #  37
 #  38
   print('Please enter a number from 1 to 12.') #  39
 #  40
 #  41
def getCalendarFor(year, month): #  42
   calText = ''  # calText will contain the string of our calendar. #  43
 #  44
   # Put the month and year at the top of the calendar: #  45
   calText += (' ' * 34) + MONTHS[month - 1] + ' ' + str(year) + '\n' #  46
 #  47
   # Add the days of the week labels to the calendar: #  48
   # (!) Try changing this to abbreviations: SUN, MON, TUE, etc. #  49
   calText += '...Sunday.....Monday....Tuesday...Wednesday...Thursday....Friday....Saturday..\n' #  50
 #  51
   # The horizontal line string that separate weeks: #  52
   weekSeparator = ('+----------' * 7) + '+\n' #  53
 #  54
   # The blank rows have ten spaces in between the | day separators: #  55
   blankRow = ('|          ' * 7) + '|\n' #  56
 #  57
   # Get the first date in the month. (The datetime module handles all #  58
   # the complicated calendar stuff for us here.) #  59
   currentDate = datetime.date(year, month, 1) #  60
 #  61
   # Roll back currentDate until it is Sunday. (weekday() returns 6 #  62
   # for Sunday, not 0.) #  63
   while currentDate.weekday() != 6: #  64
       currentDate -= datetime.timedelta(days=1) #  65
 #  66
   while True:  # Loop over each week in the month. #  67
       calText += weekSeparator #  68
 #  69
       # dayNumberRow is the row with the day number labels: #  70
       dayNumberRow = '' #  71
       for i in range(7): #  72
           dayNumberLabel = str(currentDate.day).rjust(2) #  73
           dayNumberRow += '|' + dayNumberLabel + (' ' * 8) #  74
           currentDate += datetime.timedelta(days=1) # Go to next day. #  75
       dayNumberRow += '|\n'  # Add the vertical line after Saturday. #  76
 #  77
       # Add the day number row and 3 blank rows to the calendar text. #  78
       calText += dayNumberRow #  79
       for i in range(3):  # (!) Try changing the 4 to a 5 or 10. #  80
           calText += blankRow #  81
 #  82
       # Check if we're done with the month: #  83
       if currentDate.month != month: #  84
           break #  85
 #  86
   # Add the horizontal line at the very bottom of the calendar. #  87
   calText += weekSeparator #  88
   return calText #  89
 #  90
 #  91
calText = getCalendarFor(year, month) #  92
print(calText)  # Display the calendar. #  93
 #  94
# Save the calendar to a text file: #  95
calendarFilename = 'calendar_{}_{}.txt'.format(year, month) #  96
with open(calendarFilename, 'w') as fileObj: #  97
   fileObj.write(calText) #  98
 #  99
print('Saved to ' + calendarFilename)  # 100
```
