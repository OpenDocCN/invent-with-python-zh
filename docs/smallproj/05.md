21 点，也称为 21 点，是一种纸牌游戏，玩家试图在不超过 21 点的情况下尽可能接近 21 点。这个程序使用用文本字符绘制的图像，称为 *ASCII 艺术画*。美国信息交换标准码(ASCII)是 Unicode 取代之前计算机使用的文本字符到数字代码的映射。本程序中的扑克牌是 ASCII 艺术的一个例子:

```py
 ___   ___
|A  | |10 |
| ♣ | | ♦ |
|__A| |_10|
```

你可以在[`en.wikipedia.org/wiki/Blackjack`](https://en.wikipedia.org/wiki/Blackjack)找到其他规则和这个纸牌游戏的历史。

## 它是如何工作的

你的键盘上没有牌组符号，这就是为什么我们调用`chr()`函数来创建它们。传递给`chr()`的整数被称为 Unicode *码位*，这是根据 Unicode 标准标识字符的唯一数字。Unicode 经常被误解。然而，Ned Batchelder 的 2012 年 PyCon US talk“实用 Unicode，或者我如何停止痛苦？”是对 Unicode 的极好介绍，您可以在`youtu.be/sgHbC6udIqc`找到它。附录 B 给出了可以在 Python 程序中使用的 Unicode 字符的完整列表。

```py
"""Blackjack, by Al Sweigart email@protected #  1
The classic card game also known as 21\. (This version doesn't have #  2
splitting or insurance.) #  3
More info at: https://en.wikipedia.org/wiki/Blackjack #  4
This code is available at https://nostarch.com/big-book-small-python-programming #  5
Tags: large, game, card game""" #  6
 #  7
import random, sys #  8
 #  9
# Set up the constants: #  10
HEARTS   = chr(9829) # Character 9829 is '♥'. #  11
DIAMONDS = chr(9830) # Character 9830 is '♦'. #  12
SPADES   = chr(9824) # Character 9824 is '♠'. #  13
CLUBS    = chr(9827) # Character 9827 is '♣'. #  14
# (A list of chr codes is at https://inventwithpython.com/charactermap) #  15
BACKSIDE = 'backside' #  16
 #  17
 #  18
def main(): #  19
   print('''Blackjack, by Al Sweigart email@protected #  20
 #  21
   Rules: #  22
     Try to get as close to 21 without going over. #  23
     Kings, Queens, and Jacks are worth 10 points. #  24
     Aces are worth 1 or 11 points. #  25
     Cards 2 through 10 are worth their face value. #  26
     (H)it to take another card. #  27
     (S)tand to stop taking cards. #  28
     On your first play, you can (D)ouble down to increase your bet #  29
     but must hit exactly one more time before standing. #  30
     In case of a tie, the bet is returned to the player. #  31
     The dealer stops hitting at 17.''') #  32
 #  33
   money = 5000 #  34
   while True:  # Main game loop. #  35
       # Check if the player has run out of money: #  36
       if money <= 0: #  37
           print("You're broke!") #  38
           print("Good thing you weren't playing with real money.") #  39
           print('Thanks for playing!') #  40
           sys.exit() #  41
 #  42
       # Let the player enter their bet for this round: #  43
       print('Money:', money) #  44
       bet = getBet(money) #  45
 #  46
       # Give the dealer and player two cards from the deck each: #  47
       deck = getDeck() #  48
       dealerHand = [deck.pop(), deck.pop()] #  49
       playerHand = [deck.pop(), deck.pop()] #  50
 #  51
       # Handle player actions: #  52
       print('Bet:', bet) #  53
       while True:  # Keep looping until player stands or busts. #  54
           displayHands(playerHand, dealerHand, False) #  55
           print() #  56
 #  57
           # Check if the player has bust: #  58
           if getHandValue(playerHand) > 21: #  59
               break #  60
 #  61
           # Get the player's move, either H, S, or D: #  62
           move = getMove(playerHand, money - bet) #  63
 #  64
           # Handle the player actions: #  65
           if move == 'D': #  66
               # Player is doubling down, they can increase their bet: #  67
               additionalBet = getBet(min(bet, (money - bet))) #  68
               bet += additionalBet #  69
               print('Bet increased to {}.'.format(bet)) #  70
               print('Bet:', bet) #  71
 #  72
           if move in ('H', 'D'): #  73
               # Hit/doubling down takes another card. #  74
               newCard = deck.pop() #  75
               rank, suit = newCard #  76
               print('You drew a {} of {}.'.format(rank, suit)) #  77
               playerHand.append(newCard) #  78
 #  79
               if getHandValue(playerHand) > 21: #  80
                   # The player has busted: #  81
                   continue #  82
 #  83
           if move in ('S', 'D'): #  84
               # Stand/doubling down stops the player's turn. #  85
               break #  86
 #  87
       # Handle the dealer's actions: #  88
       if getHandValue(playerHand) <= 21: #  89
           while getHandValue(dealerHand) < 17: #  90
               # The dealer hits: #  91
               print('Dealer hits...') #  92
               dealerHand.append(deck.pop()) #  93
               displayHands(playerHand, dealerHand, False) #  94
 #  95
               if getHandValue(dealerHand) > 21: #  96
                   break  # The dealer has busted. #  97
               input('Press Enter to continue...') #  98
               print('\n\n') #  99
 # 100
        # Show the final hands: # 101
        displayHands(playerHand, dealerHand, True) # 102
 # 103
        playerValue = getHandValue(playerHand) # 104
        dealerValue = getHandValue(dealerHand) # 105
        # Handle whether the player won, lost, or tied: # 106
        if dealerValue > 21: # 107
            print('Dealer busts! You win ${}!'.format(bet)) # 108
            money += bet # 109
        elif (playerValue > 21) or (playerValue < dealerValue): # 110
            print('You lost!') # 111
            money -= bet # 112
        elif playerValue > dealerValue: # 113
            print('You won ${}!'.format(bet)) # 114
            money += bet # 115
        elif playerValue == dealerValue: # 116
            print('It\'s a tie, the bet is returned to you.') # 117
 # 118
        input('Press Enter to continue...') # 119
        print('\n\n') # 120
 # 121
 # 122
def getBet(maxBet): # 123
    """Ask the player how much they want to bet for this round.""" # 124
    while True:  # Keep asking until they enter a valid amount. # 125
        print('How much do you bet? (1-{}, or QUIT)'.format(maxBet)) # 126
        bet = input('> ').upper().strip() # 127
        if bet == 'QUIT': # 128
            print('Thanks for playing!') # 129
            sys.exit() # 130
 # 131
        if not bet.isdecimal(): # 132
            continue  # If the player didn't enter a number, ask again. # 133
 # 134
        bet = int(bet) # 135
        if 1 <= bet <= maxBet: # 136
            return bet  # Player entered a valid bet. # 137
 # 138
 # 139
def getDeck(): # 140
    """Return a list of (rank, suit) tuples for all 52 cards.""" # 141
    deck = [] # 142
    for suit in (HEARTS, DIAMONDS, SPADES, CLUBS): # 143
        for rank in range(2, 11): # 144
            deck.append((str(rank), suit))  # Add the numbered cards. # 145
        for rank in ('J', 'Q', 'K', 'A'): # 146
            deck.append((rank, suit))  # Add the face and ace cards. # 147
    random.shuffle(deck) # 148
    return deck # 149
 # 150
 # 151
def displayHands(playerHand, dealerHand, showDealerHand): # 152
    """Show the player's and dealer's cards. Hide the dealer's first # 153
    card if showDealerHand is False.""" # 154
    print() # 155
    if showDealerHand: # 156
        print('DEALER:', getHandValue(dealerHand)) # 157
        displayCards(dealerHand) # 158
    else: # 159
        print('DEALER: ???') # 160
        # Hide the dealer's first card: # 161
        displayCards([BACKSIDE] + dealerHand[1:]) # 162
 # 163
    # Show the player's cards: # 164
    print('PLAYER:', getHandValue(playerHand)) # 165
    displayCards(playerHand) # 166
 # 167
 # 168
def getHandValue(cards): # 169
    """Returns the value of the cards. Face cards are worth 10, aces are # 170
    worth 11 or 1 (this function picks the most suitable ace value).""" # 171
    value = 0 # 172
    numberOfAces = 0 # 173
 # 174
    # Add the value for the non-ace cards: # 175
    for card in cards: # 176
        rank = card[0]  # card is a tuple like (rank, suit) # 177
        if rank == 'A': # 178
            numberOfAces += 1 # 179
        elif rank in ('K', 'Q', 'J'):  # Face cards are worth 10 points. # 180
            value += 10 # 181
        else: # 182
            value += int(rank)  # Numbered cards are worth their number. # 183
 # 184
    # Add the value for the aces: # 185
    value += numberOfAces  # Add 1 per ace. # 186
    for i in range(numberOfAces): # 187
        # If another 10 can be added without busting, do so: # 188
        if value + 10 <= 21: # 189
            value += 10 # 190
 # 191
    return value # 192
 # 193
 # 194
def displayCards(cards): # 195
    """Display all the cards in the cards list.""" # 196
    rows = ['', '', '', '', '']  # The text to display on each row. # 197
 # 198
    for i, card in enumerate(cards): # 199
        rows[0] += ' ___  '  # Print the top line of the card. # 200
        if card == BACKSIDE: # 201
            # Print a card's back: # 202
            rows[1] += '|## | ' # 203
            rows[2] += '|###| ' # 204
            rows[3] += '|_##| ' # 205
        else: # 206
            # Print the card's front: # 207
            rank, suit = card  # The card is a tuple data structure. # 208
            rows[1] += '|{} | '.format(rank.ljust(2)) # 209
            rows[2] += '| {} | '.format(suit) # 210
            rows[3] += '|_{}| '.format(rank.rjust(2, '_')) # 211
 # 212
    # Print each row on the screen: # 213
    for row in rows: # 214
        print(row) # 215
 # 216
 # 217
def getMove(playerHand, money): # 218
    """Asks the player for their move, and returns 'H' for hit, 'S' for # 219
    stand, and 'D' for double down.""" # 220
    while True:  # Keep looping until the player enters a correct move. # 221
        # Determine what moves the player can make: # 222
        moves = ['(H)it', '(S)tand'] # 223
 # 224
        # The player can double down on their first move, which we can # 225
        # tell because they'll have exactly two cards: # 226
        if len(playerHand) == 2 and money > 0: # 227
            moves.append('(D)ouble down') # 228
 # 229
        # Get the player's move: # 230
        movePrompt = ', '.join(moves) + '> ' # 231
        move = input(movePrompt).upper() # 232
        if move in ('H', 'S'): # 233
            return move  # Player has entered a valid move. # 234
        if move == 'D' and '(D)ouble down' in moves: # 235
            return move  # Player has entered a valid move. # 236
 # 237
 # 238
# If the program is run (instead of imported), run the game: # 239
if __name__ == '__main__': # 240
    main()  # 241
```
