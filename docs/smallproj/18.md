这个数学测验程序掷出两到六个骰子，你必须尽快把它们的边加起来。但是这个程序不仅仅是自动的闪存卡；它将骰子的正面画到屏幕上随机的地方。当你练习算术时，ASCII-art 方面增加了一个有趣的转折。

## 它是如何工作的

屏幕上的骰子由存储在`canvas`变量中的字典表示。在 Python 中，元组类似于列表，但是它们的内容不能改变。该字典的关键字是标记骰子左上角位置的`(x, y)`元组，而值是`ALL_DICE`中的“骰子元组”之一。您可以在第 28 到 80 行中看到，每个骰子元组包含一个字符串列表，它以图形方式表示一个可能的骰子面，以及骰子面上有多少点数的整数。该程序使用这些信息来显示骰子并计算它们的总和。

第 174 到 177 行将`canvas`字典中的数据呈现在屏幕上，其方式类似于项目 13“康威的生命游戏”在屏幕上呈现单元格的方式。

```py
"""Dice Math, by Al Sweigart email@protected #  1
A flash card addition game where you sum the total on random dice rolls. #  2
View this code at https://nostarch.com/big-book-small-python-projects #  3
Tags: large, artistic, game, math""" #  4
 #  5
import random, time #  6
 #  7
# Set up the constants: #  8
DICE_WIDTH = 9 #  9
DICE_HEIGHT = 5 #  10
CANVAS_WIDTH = 79 #  11
CANVAS_HEIGHT = 24 - 3  # -3 for room to enter the sum at the bottom. #  12
 #  13
# The duration is in seconds: #  14
QUIZ_DURATION = 30  # (!) Try changing this to 10 or 60. #  15
MIN_DICE = 2  # (!) Try changing this to 1 or 5. #  16
MAX_DICE = 6  # (!) Try changing this to 14. #  17
 #  18
# (!) Try changing these to different numbers: #  19
REWARD = 4  # (!) Points awarded for correct answers. #  20
PENALTY = 1  # (!) Points removed for incorrect answers. #  21
# (!) Try setting PENALTY to a negative number to give points for #  22
# wrong answers! #  23
 #  24
# The program hangs if all of the dice can't fit on the screen: #  25
assert MAX_DICE <= 14 #  26
 #  27
D1 = (['+-------+', #  28
      '|       |', #  29
      '|   O   |', #  30
      '|       |', #  31
      '+-------+'], 1) #  32
 #  33
D2a = (['+-------+', #  34
       '| O     |', #  35
       '|       |', #  36
       '|     O |', #  37
       '+-------+'], 2) #  38
 #  39
D2b = (['+-------+', #  40
       '|     O |', #  41
       '|       |', #  42
       '| O     |', #  43
       '+-------+'], 2) #  44
 #  45
D3a = (['+-------+', #  46
       '| O     |', #  47
       '|   O   |', #  48
       '|     O |', #  49
       '+-------+'], 3) #  50
 #  51
D3b = (['+-------+', #  52
       '|     O |', #  53
       '|   O   |', #  54
       '| O     |', #  55
       '+-------+'], 3) #  56
 #  57
D4 = (['+-------+', #  58
      '| O   O |', #  59
      '|       |', #  60
      '| O   O |', #  61
      '+-------+'], 4) #  62
 #  63
D5 = (['+-------+', #  64
      '| O   O |', #  65
      '|   O   |', #  66
      '| O   O |', #  67
      '+-------+'], 5) #  68
 #  69
D6a = (['+-------+', #  70
       '| O   O |', #  71
       '| O   O |', #  72
       '| O   O |', #  73
       '+-------+'], 6) #  74
 #  75
D6b = (['+-------+', #  76
       '| O O O |', #  77
       '|       |', #  78
       '| O O O |', #  79
       '+-------+'], 6) #  80
 #  81
ALL_DICE = [D1, D2a, D2b, D3a, D3b, D4, D5, D6a, D6b] #  82
 #  83
print('''Dice Math, by Al Sweigart email@protected #  84
 #  85
Add up the sides of all the dice displayed on the screen. You have #  86
{} seconds to answer as many as possible. You get {} points for each #  87
correct answer and lose {} point for each incorrect answer. #  88
'''.format(QUIZ_DURATION, REWARD, PENALTY)) #  89
input('Press Enter to begin...') #  90
 #  91
# Keep track of how many answers were correct and incorrect: #  92
correctAnswers = 0 #  93
incorrectAnswers = 0 #  94
startTime = time.time() #  95
while time.time() < startTime + QUIZ_DURATION:  # Main game loop. #  96
   # Come up with the dice to display: #  97
   sumAnswer = 0 #  98
   diceFaces = [] #  99
    for i in range(random.randint(MIN_DICE, MAX_DICE)): # 100
        die = random.choice(ALL_DICE) # 101
        # die[0] contains the list of strings of the die face: # 102
        diceFaces.append(die[0]) # 103
        # die[1] contains the integer number of pips on the face: # 104
        sumAnswer += die[1] # 105
 # 106
    # Contains (x, y) tuples of the top-left corner of each die. # 107
    topLeftDiceCorners = [] # 108
 # 109
    # Figure out where dice should go: # 110
    for i in range(len(diceFaces)): # 111
        while True: # 112
            # Find a random place on the canvas to put the die: # 113
            left = random.randint(0, CANVAS_WIDTH  - 1 - DICE_WIDTH) # 114
            top  = random.randint(0, CANVAS_HEIGHT - 1 - DICE_HEIGHT) # 115
 # 116
            # Get the x, y coordinates for all four corners: # 117
            #      left # 118
            #      v # 119
            #top > +-------+ ^ # 120
            #      | O     | | # 121
            #      |   O   | DICE_HEIGHT (5) # 122
            #      |     O | | # 123
            #      +-------+ v # 124
            #      <-------> # 125
            #      DICE_WIDTH (9) # 126
            topLeftX = left # 127
            topLeftY = top # 128
            topRightX = left + DICE_WIDTH # 129
            topRightY = top # 130
            bottomLeftX = left # 131
            bottomLeftY = top + DICE_HEIGHT # 132
            bottomRightX = left + DICE_WIDTH # 133
            bottomRightY = top + DICE_HEIGHT # 134
 # 135
            # Check if this die overlaps with previous dice. # 136
            overlaps = False # 137
            for prevDieLeft, prevDieTop in topLeftDiceCorners: # 138
                prevDieRight = prevDieLeft + DICE_WIDTH # 139
                prevDieBottom = prevDieTop + DICE_HEIGHT # 140
                # Check each corner of this die to see if it is inside # 141
                # of the area the previous die: # 142
                for cornerX, cornerY in ((topLeftX, topLeftY), # 143
                                         (topRightX, topRightY), # 144
                                         (bottomLeftX, bottomLeftY), # 145
                                         (bottomRightX, bottomRightY)): # 146
                    if (prevDieLeft <= cornerX < prevDieRight # 147
                        and prevDieTop <= cornerY < prevDieBottom): # 148
                            overlaps = True # 149
            if not overlaps: # 150
                # It doesn't overlap, so we can put it here: # 151
                topLeftDiceCorners.append((left, top)) # 152
                break # 153
 # 154
    # Draw the dice on the canvas: # 155
 # 156
    # Keys are (x, y) tuples of ints, values the character at that # 157
    # position on the canvas: # 158
    canvas = {} # 159
    # Loop over each die: # 160
    for i, (dieLeft, dieTop) in enumerate(topLeftDiceCorners): # 161
        # Loop over each character in the die's face: # 162
        dieFace = diceFaces[i] # 163
        for dx in range(DICE_WIDTH): # 164
            for dy in range(DICE_HEIGHT): # 165
                # Copy this character to the correct place on the canvas: # 166
                canvasX = dieLeft + dx # 167
                canvasY = dieTop + dy # 168
                # Note that in dieFace, a list of strings, the x and y # 169
                # are swapped: # 170
                canvas[(canvasX, canvasY)] = dieFace[dy][dx] # 171
 # 172
    # Display the canvas on the screen: # 173
    for cy in range(CANVAS_HEIGHT): # 174
        for cx in range(CANVAS_WIDTH): # 175
            print(canvas.get((cx, cy), ' '), end='') # 176
        print()  # Print a newline. # 177
 # 178
    # Let the player enter their answer: # 179
    response = input('Enter the sum: ').strip() # 180
    if response.isdecimal() and int(response) == sumAnswer: # 181
        correctAnswers += 1 # 182
    else: # 183
        print('Incorrect, the answer is', sumAnswer) # 184
        time.sleep(2) # 185
        incorrectAnswers += 1 # 186
 # 187
# Display the final score: # 188
score = (correctAnswers * REWARD) - (incorrectAnswers * PENALTY) # 189
print('Correct:  ', correctAnswers) # 190
print('Incorrect:', incorrectAnswers) # 191
print('Score:    ', score)  # 192
```
